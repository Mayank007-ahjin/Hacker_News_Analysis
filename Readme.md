# 📊 Hacker News Data Analysis

## 📌 Overview

This project analyzes Hacker News posts to find patterns in engagement (points & comments).

## 🔍 Key Insights

- Best time to post based on average points
- Relationship between comments and points
- Data cleaning and preprocessing

## 🛠️ Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn

## 📂 Structure

- data/ → dataset
- notebooks/ → original Jupyter notebook
- src/ → analysis script
- outputs/ → saved visualizations

## ▶️ Run Project

```bash
pip install -r requirements.txt
python src/analysis.py
